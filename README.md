# GiftLink

A platform connecting donors with recipients through direct 1:1 essential item donations.

## Features

- Direct 1:1 donation matching system
- AI-generated character images for privacy
- Essential items focused donation
- Real-time status tracking

## Tech Stack

- Next.js 13+
- TypeScript
- Prisma
- PostgreSQL
- OpenAI API
- Tailwind CSS

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/action5861/giftlink.git
```

2. Install dependencies
```bash
cd giftlink
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
# Fill in your environment variables in .env
```

4. Run the development server
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/app` - Next.js application routes
- `/components` - Reusable UI components
- `/lib` - Utility functions and configurations
- `/prisma` - Database schema and migrations
- `/public` - Static assets
- `/types` - TypeScript type definitions
