# GiftLink

A platform connecting donors with recipients through direct 1:1 essential item donations.

## Features

- Direct 1:1 donation matching system
- AI-generated character images for privacy
- Essential items focused donation
- Real-time status tracking

## Tech Stack

- Next.js 13+
- TypeScript
- Prisma
- PostgreSQL
- OpenAI API
- Tailwind CSS
